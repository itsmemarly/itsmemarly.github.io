Festival Project

This project was a collaboration with my project group at NHL Stenden University. The assignment came to us through the Digitale Werkplaats on behalf of Survento. Survento is a fanfare that organizes various festivals. As students of the HBO-ICT program, we had the opportunity to design and implement a solution for Survento's festivals, working together with other students.

The main goal of the Festival Project was to create an intuitive and engaging platform where festival-goers could view the program.

Throughout the development process, we used various technologies, including PHP, HTML, CSS, and JavaScript, to build a dynamic and user-friendly website. The backend, powered by PHP, managed users & sponsors, while HTML and CSS were used to create intuitive user interfaces and visually appealing designs. JavaScript added interactivity and enhanced the overall user experience.

The project provided an excellent opportunity for teamwork, collaboration, and problem-solving as we worked closely with other students from the HBO-ICT program to bring the Festival Project to life. By utilizing version control with Git and hosting the project on GitHub, we ensured smooth and efficient collaboration.

The end result was a successful and fully functional web application that exceeded Survento's expectations. The Festival Project not only met the client's requirements but also provided a seamless and enjoyable festival experience for participants.

The purpose of this project was to create a digital program booklet for the festival.

Web Development Project "Stripboekenverzameling"

The Web Development Project "Stripboekenverzameling" was a collaborative effort with fellow students from the HBO-ICT program. Created for the Web Development course, the project aimed to address our client's need for an organized comic book collection. To achieve this, we implemented a robust database system.

Our client desired a user-friendly system that allowed them to add, edit, and remove books from their collection seamlessly. Each book's information, including essential details and its location, was stored in the database for easy access and management.

The project showcased our team's expertise in web development and database design, providing an efficient solution for our client's comic book collection organization. The result was a well-structured system that met our client's needs.

Language Learning Tool

I have a strong connection with people who come to the Netherlands and wish to learn the Dutch language. Upon interacting with them, I discovered a common challenge they faced - the lack of a simple tool to enhance their vocabulary. Determined to address this issue, I embarked on building a system that would cater to their needs and provide a solution for improving their language skills.

The result was a comprehensive language learning application that allows individuals to create their own customized wordlists and test their knowledge through interactive flashcards. By offering this platform, language learners can practice and expand their vocabulary in an efficient and enjoyable manner.

This application not only facilitates learning but also fosters a sense of community among language learners. Users have the freedom to import and export wordlists, enabling them to share and exchange knowledge with one another. This collaborative aspect enhances the overall learning experience and encourages collective language development.

While initially designed for language learning, the versatility of this system opens up possibilities for use in various academic subjects, such as mathematics, biology, and history. The tool can be adapted to cater to a wide range of educational needs and foster effective learning in different disciplines.

Through this project, I aim to empower language learners and students in diverse fields, enabling them to master new subjects with confidence and proficiency. The application's user-friendly approach and focus on collaboration promote a positive and enriching learning environment for all users.

This journey of creating a language learning tool has not only enriched the lives of those seeking to master Dutch but also has the potential to impact learners worldwide. By promoting knowledge exchange and fostering a love for learning, this application contributes to the growth and enrichment of minds and cultures across the globe.
